# 🎮 SquadHub – Rede Social Gamer

O SquadHub é uma aplicação desenvolvida como projeto acadêmico, com o objetivo de simular uma rede social voltada para o público gamer. A plataforma permite interação entre usuários por meio de publicações, conexões e comunicação direta.

---

## 🚀 Funcionalidades

- Cadastro e login de usuários  
- Edição de perfil  
- Publicação de conteúdos  
- Curtidas e comentários em postagens  
- Sistema de amizade (envio, aceite e recusa de solicitações)  
- Visualização da rede de amigos  
- Envio de mensagens privadas (direct)  

---

## 🧠 Desafio Técnico

O principal desafio deste projeto foi implementar toda a lógica de armazenamento de dados utilizando estruturas de dados em memória, sem o uso de banco de dados.

Os dados da aplicação são manipulados por meio de listas, explorando conceitos como:

- Estruturas de dados  
- Organização e manipulação de informações  
- Lógica de programação aplicada  
- Simulação de persistência em memória  

---

## 🛠️ Tecnologias Utilizadas

- C#  
- Windows Forms  
- Visual Studio 2019  

---

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com foco no aprendizado de lógica de programação e estruturas de dados, aplicados na construção de uma aplicação completa com múltiplas funcionalidades e interação entre usuários.
